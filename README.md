### 部署方法

1. 提交内容到 github
2. 本地验证 `yarn run start`
3. 部署到 github page `yarn deploy`
4. 需要实现配置好 [docusaurus.config.js](https://docusaurus.org.cn/docs/deployment)
5. 登录 `https://banrieen.github.io`
